## Introduction

STDIO MCP server for searching through sessions of the ADC 2026 conference.

## Tech Stack

* .NET 10 with C#

## Documentation

Whenever you need up-to-date docs for Microsoft-related technologies including C# and .NET, use the `MS-Learn` MCP server. Do NOT use Context7 in this case. Only read Context7 if you did not find the answer in `MS-Learn`.

If you build an MCP server, read the `mcp-builder` skill first. It contains general information about the protocol, design principles, and best practices.

For any other technologies INCLUDING the MCP C# SDK, use Context7. The library ID for the MCP C# SDK is `modelcontextprotocol/csharp-sdk`.
